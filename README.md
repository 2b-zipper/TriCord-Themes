# TriCord Themes

A repository of custom themes for [TriCord](https://github.com/2b-zipper/TriCord), a Discord client for the Nintendo 3DS.

## Installation

1. Download a theme `.json` file from this repository.
2. Copy the file to `/3ds/TriCord/themes/` on your SD card.
3. Launch TriCord and open Theme Manager from the settings menu.
4. Select a theme to apply it. Press A again on the active theme to disable it.
