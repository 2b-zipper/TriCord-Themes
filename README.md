# TriCord Themes

A repository of custom themes for TriCord, a Discord client for the Nintendo 3DS.

## Installation

### Manual Installation
1. Download a theme `.json` file from this repository.
2. Copy the file to the `/3ds/TriCord/themes/` directory on your SD card.
3. Launch TriCord and go to **Settings > Manage Custom Themes** to select and enable your theme.

## Available Themes

| Theme Name | Author | Description |
| :--- | :--- | :--- |
| **Dracula** | 2b-zipper | A vibrant and modern dark theme. |
| **Pure Black** | 2b-zipper | A pure black theme. |

## Creating New Themes
To create your own theme, you can use any existing `.json` file as a template. New themes placed in the `/3ds/TriCord/themes/` directory will be automatically recognized by the application.
